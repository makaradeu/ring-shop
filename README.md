# 💍 Ring Shop

A unique ring selling web application that helps reduce breakups 
by connecting couples through meaningful ring purchases.

## Features
- 💍 Browse and buy unique rings
- 👤 User registration and login
- 💑 Couple linking system
- 📅 Anniversary date reminders
- 💌 Love messages with ring purchases

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MySQL

## How to Run

### Backend
cd backend
npm install
npx nodemon index.js

### Frontend
cd frontend
npm install
npm start
```

Then push it:
```
git add .
git commit -m "add README"
git push
